TrueFalseAPI Docs

The TrueFalseAPI is a simple API that provides access to a collection of true/false statements across different categories. The API is free to use and does not require an API key. See the documentation below on making requests to the api.

BaseURL

The base URL for the API is

https://true-false-api.vercel.app

Endpoints

GET /all

Returns all the true/false statements available in the API.

Example Request

GET https://true-false-api.vercel.app/all

Example Response

[
    {
    "id": 1,
    "statement": "Bananas are berries",
    "is_true": true,
    "category": "food"
    },
    {
    "id": 2,
    "statement": "Tomatoes are fruits",
    "is_true": true,
    "category": "food"
    },
    ...
]

GET /random

Returns a random true/false statement from the API.

Example Request

GET https://true-false-api.vercel.app/random

Example Response

{
"id": 1,
"statement": "Bananas are berries",
"is_true": true,    
"category": "food"
}

GET /{category}

Returns all the true/false statements for a specific category. The available categories are:

  • animals
  • food
  • history
  • science
  • technology
  • entertainment

Example Request

GET https://true-false-api.vercel.app/category/food

Example Response

[
    {
    "id": 1,
    "statement": "Bananas are berries",
    "is_true": true,
    "category": "food"
    },
    {
    "id": 2,
    "statement": "Tomatoes are fruits",
    "is_true": true,
    "category": "food"
    },
    ...
]

GET/{category}/random

Returns a random true/false statement from a specific category. The available categories are:

  • animal
  • food
  • history
  • science
  • technology
  • entertainment

Example Request

GET https://true-false-api.vercel.app/food/random

Example Response

{
"id": 1,
"statement": "Bananas are berries",
"is_true": true,
"category": "food"
}

Example Response

{
